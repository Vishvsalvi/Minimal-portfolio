## Clone the repositotry
```bash
git clone https://github.com/Vishvsalvi/Minimal-portfolio
```

## Navigate to the project directory
```bash
cd Minimal-portfolio
```

## Install the packages
```bash
npm install
```

## Run it
```bash
npm run dev
```
